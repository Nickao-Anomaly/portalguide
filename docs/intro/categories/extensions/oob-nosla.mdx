# OOB No SLA

OOB No SLA, meaning "out of bounds, no save/load abuse", is an extension category that allows you to follow the
[rules of the "out of bounds" category](/categories/oob), but with the added restriction of denying the use of
[save/load abuse](/terms#save-load-abuse).
